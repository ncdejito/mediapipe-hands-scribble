# mediapipe-hands-scribble

Drawing with fingers using MediaPipe Hands.

![](demo.gif)
*Me casting a wind spell*

## Usage
1. Install vlc
2. Create environment
```
python -m venv venv/
source venv/bin/activate
```
3. Run 
```
python main.py
```
